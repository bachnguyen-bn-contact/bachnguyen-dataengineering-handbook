# Exam overview

**DP-700** earns you the *Microsoft Certified: Fabric Data Engineer Associate* credential. It validates that you can design, implement, secure, monitor and optimise data engineering solutions on Microsoft Fabric.

## The shape of the exam

| | |
|---|---|
| Format | Proctored, role-based; can include case studies and interactive items |
| Length | ~100 minutes |
| Questions | Roughly 40–60 |
| Passing score | 700 out of 1000 (scaled scoring) |
| Prerequisites | None, but you'll need real SQL, PySpark and KQL ability |
| Renewal | Annually, free, via an online assessment on Microsoft Learn |
| In-exam help | Microsoft Learn docs are available in a split-screen during the exam |

Because scoring is scaled, the smart move is **broad coverage across all three domains** rather than chasing a specific raw number of correct questions.

## It replaced DP-203

DP-700 is the Fabric-era successor for data engineers. The older Azure Data Engineer exam, **DP-203, retired in 2025**, and Microsoft now steers new candidates to Fabric role-based paths. The platform underneath is different: OneLake instead of raw Data Lake Storage, and KQL alongside SQL.

If you're sitting on half-finished DP-203 prep, set it aside — that exam no longer exists.

## A moving target

Fabric updates monthly, and the exam's *"skills measured as of"* date moves with it. Always open the [official study guide](/dp700/official-resources) and check that date before you start, because new (sometimes preview) features can appear on the exam.
