# CI/CD, monitoring & decisions

## Git integration & deployment pipelines [Intermediate]

Connect a workspace to Git for version control of Fabric items, and use **deployment pipelines** to promote content across Dev, Test and Prod with controlled, repeatable releases.

## Monitoring [Intermediate]

Use the Monitoring hub to track pipeline and item runs, spot failures, and watch capacity usage. Wire alerts so problems find you, not the other way around.

## When to use what [Intermediate]

| Goal | Reach for |
|---|---|
| Big file transforms | PySpark / Spark SQL |
| Set-based analytics with writes | Warehouse (T-SQL) |
| Streaming / telemetry | Real-Time Intelligence (KQL) |
| Low-code shaping | Dataflow Gen2 |
| Report metrics | Power BI (DAX) |
