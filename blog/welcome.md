# Why I built this site

Hi, I'm Bach.

I'm a data engineer who enjoys building data platforms, solving messy data problems, and continuously learning new technologies. Most of my work revolves around Microsoft Fabric, Databricks, cloud data platforms, SQL, Python, and Spark.

I built this site for a simple reason: I wanted a place to document what I'm learning.

Over time, I found myself saving useful resources everywhere—browser bookmarks, OneNote pages, random documents, chat conversations, and screenshots. Whenever I needed something again, I would spend more time searching for it than actually using it.

This website is my attempt to fix that.

You'll find:

* **Fabric Handbook** — practical notes, concepts, and best practices for Microsoft Fabric, written from the perspective of someone using it every day.
* **DP-700 Handbook** — study notes, resources, and explanations for the Microsoft Fabric Data Engineer certification.
* **Blog** — thoughts, lessons learned, project write-ups, and anything interesting I come across along the way.

More than anything, this site is a living notebook. As I grow as a data engineer, I'll continue adding new ideas, documenting lessons learned, and sharing things I wish I had known earlier.

If you happen to stumble across this post and find something useful, I'd love to connect. Feel free to reach out on [LinkedIn](/resume), or [book a 1-on-1](/resume) if you'd like to chat about data engineering, Microsoft Fabric, career growth, projects, certifications, or anything else data-related.

Thanks for stopping by, and welcome aboard.
